내부 클래스

1. 인스턴스 내부 클래스
	- instance : 실체 (생성된 객체 = 객체)
	- 멤버 변수(인스턴스 변수), 인스턴스 메서드 

2. 정적 내부 클래스
	- static : 정적 변수, 정적 메서드

3. 지역 내부 클래스
	- 함수(메서드)

	- 추상 클래스, 인터페이스 객체 생성 조건
		- 특정 환경(지역 내부, 멤버 변수)
		- 미 구현된 메서드를 구현
		- Runnable - run

4. 익명 내부 클래스

인터페이스, 추상 클래스
	1) 지역 내부(함수, 메서드)
	2) 멤버 변수
	3) 미 구현된 메서드 구현
